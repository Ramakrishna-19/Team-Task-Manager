# Team Task Manager

## Live URL
https://team-task-manager-amber.vercel.app/

## Backend URL
https://team-task-manager-s7nh.onrender.com

## Features
- User Authentication (Login/Register)
- Role-based access (Admin / Member)
- Project management
- Task assignment & tracking
- Dashboard overview
- Admin email control

## Tech Stack
- Frontend: React (Vite)
- Backend: Node.js, Express
- Database: MongoDB Atlas
- Deployment: Vercel + Render

## Setup Instructions

### Backend
cd server
npm install
npm run dev

### Frontend
cd client
npm install
npm run dev
